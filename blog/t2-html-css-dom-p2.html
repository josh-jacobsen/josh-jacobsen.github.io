What is grid based design?

Grid based design describes the process of using a grid to layout the content of a website. This provides a good visual and structural balance when creating a website. They allow web designers to systimatically craft well-organized layouts, and provide the user with a predictable pattern to follow as they use the site.

What is all the hype about responsive webpages?

Responsive webpages change to accomodate screens of different dimentions (in pixels, as oppossed to the physical size of the screen itself). A related concept is adaptive design, which is the process of ensuring websites are able to be manipulated by users on differnet devices, who may be using thumbs to navigate, as opposed to a mouse.

What is Semantic Structure?

Semantic structure uses semantic elements to give structure to the webpage that gives meaning to both the browser and the developer. The use of tags such as article, section, table, and img clearly define the content of the sections. There are benefits for search engines that are now able to correctly identify webpage content
