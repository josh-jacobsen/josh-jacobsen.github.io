Javascript Language Symbols

Parentheses

Parentheses (()) are used most often in Javascript at the end of functions. When creating a function, arguments are placed within the parentheses that can be used within the function. In the following example, the argument "event" is passed to the function in order to determine which key was pressed: 

              
    function keyUsed(event) {
      // key a
      if (event.which === 65){
          findActive("stripPlayer:1");
      }
      // key s
      else if (event.which === 83) {
          findActive("stripPlayer:2")
      }
      // key d
      else if (event.which === 68) {
          findActive("stripPlayer:3")
      }
      // key f
      else if (event.which === 70) {
          findActive("stripPlayer:4")
      }
    }

Brackets

Brackets ([]) are used when creating arrays. In the following example, a list of skills is created using brackets: 

               var skills = ["HTML", "CSS", "JavaScript", "Node.js"],

Braces

Braces ({}) are used when creating loops, functions, and objects. In the following example, braces encapsulate the actions carried out in the body of the for loop: 

              

    for (i = 0; i < game_players; i++){
        var cells = rows[i].getElementsByTagName("td");
        for (j = 0; j < cells.length; j++) {
            cells[0].className = "active";
        }
    }

Single and Double Quotes

Single and single quotes can be used almost interchangably in JavaScript, unlike in some other langages. Quotes are used to define stings. If we look at the block of code used in the brackets example, the quotation marks mean that the skills are string objects, as opposed to being variables: 

                 var skills = ["HTML", "CSS", "JavaScript", "Node.js"],
